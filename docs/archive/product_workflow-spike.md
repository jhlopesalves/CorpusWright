Open Directory
    recursively load supported files and subfolders

File Selection
    selecting one file previews one file
    selecting multiple files previews multiple files
    selection drives the preview

Original Text
    always shows original/extracted text for current selection

Processed Text
    shows processed preview for current selection after user requests it

Processing Settings
    configure deterministic cleaning rules

Preview Processed Selection
    apply settings to selected files only, for preview

Process Loaded Corpus
    apply settings to all loaded files deliberately

Export Processed Corpus
    export UTF-8 .txt files plus manifest/warnings/config
